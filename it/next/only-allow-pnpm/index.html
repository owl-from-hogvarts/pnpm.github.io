Consenti solo pnpm

Quando usi pnpm in un progetto, non vuoi che gli altri eseguano accidentalmente npm install o yarn. Per evitare che gli sviluppatori usino altri gestori di pacchetti, puoi aggiungere il seguente script preinstall nel tuo package.json:

{
"scripts": {
"preinstall": "npx only-allow pnpm"
}
}
